Cet exemple montre les possibilités d'encodage de fichier d'un format à l'autre directement sur le cloud. Il s'agit ici uniquement d'un tutoriel pour montrer la fonctionnalité et non d'un systême en production.

Ce projet se compose de plusieurs briques plus ou moins récentes permettant d'offrir une expérience utilisateur fluide et de qualité tel l'upload dynamique de fichiers. Le décodage et l'encodage de fichier se fait directement sur le serveur grace au projet FFmpeg combiné a Node.js (javascript coté serveur) et la librairie Socket.io.js pour les transferts d'informations client/serveur en temps réel.

Si ce projet vous intéresse, vous pouvez trouver le tutoriel de développement au complet ainsi que le code source du projet au bas de cette page ou bien sur mon blog technique: www.codeyourweb.org
